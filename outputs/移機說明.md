# Codex 專案移機說明

## 這個壓縮包包含什麼

這份搬家包包含目前專案資料夾內的檔案與隱藏資料夾，包括：

- `README.md`
- `AGENTS.md`
- `TODO.md`
- `PROGRESS.md`
- `materials/`
- `assets/`
- `outputs/`
- `skills/`
- `.agents/`
- `.claude/`
- `.mcp.json`
- `.git/`

## 到另一台電腦的建議順序

1. 先安裝 Codex。
2. 將壓縮包解壓縮到你想放專案的位置。
3. 用 Codex 開啟解壓縮後的專案資料夾。
4. 先閱讀 `README.md`、`TODO.md`、`PROGRESS.md`，確認目前進度。
5. 若要用 Git 同步兩台電腦，請在其中一台電腦建立第一次 commit，並推送到遠端 Git 倉庫；另一台電腦再用 Git pull 同步。

## 重要提醒

目前這個專案資料夾已經有 `.git/`，但尚未建立第一次 commit。只靠壓縮包可以搬移目前檔案，但若兩台電腦要長期同時工作，建議使用 Git 遠端倉庫同步，避免兩邊各自修改後產生衝突。

如果另一台電腦也需要相同的 Codex 全域設定或全域 skills，除了這個專案資料夾之外，還需要另外搬移或重新安裝使用者層級的 Codex 設定。
