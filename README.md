# Codex Project

This repository is a working space for Codex-assisted tasks.

## Purpose

- Collect source materials in `materials/`.
- Keep reusable or generated assets in `assets/`.
- Store final deliverables in `outputs/`.
- Track planned work in `TODO.md`.
- Track completed work and decisions in `PROGRESS.md`.

## Project Structure

```text
.
├── AGENTS.md
├── README.md
├── TODO.md
├── PROGRESS.md
├── materials/
├── assets/
└── outputs/
```

## Workflow

1. Clarify the goal and success criteria before starting work.
2. Make the smallest useful change.
3. Verify the result when possible.
4. Record meaningful progress in `PROGRESS.md`.
